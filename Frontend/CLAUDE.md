 ## About the website
This website represents an AI Services Company that helps businesses integrate artificial intelligence into their products, workflows, and operations.
The primary objective of the website is to showcase AI capabilities, demonstrate expertise, generate qualified leads, and convert visitors into potential clients.
The website should communicate professionalism, technical excellence, innovation, and trust from the moment a visitor lands on the homepage.
Use React + TailwindCSS for complete website

## Important Rules

* Do not hardcode colors.
* Do not hardcode content.
* Do not hardcode branding text.
* Choose typography, spacing, colors, and visual hierarchy automatically based on the overall design language.
* Every design decision should feel professional and enterprise-focused.
* Avoid anything flashy, childish, or visually overwhelming.

---

# Typography Requirements

Typography is extremely important.

## Heading Typography

Headings should:

* Be visually dominant.
* Have strong hierarchy.
* Use modern SaaS-style typography.
* Use appropriate font weight.
* Use proper line height.
* Use proper letter spacing.
* Scale correctly across devices.

The AI should determine:

* Font family
* Font weight
* Font size
* Line height
* Letter spacing

based on section importance.

---

## Body Typography

Body text should:

* Be highly readable.
* Maintain proper contrast with the background.
* Use comfortable line spacing.
* Follow modern UI standards.

The AI should automatically determine:

* Font size
* Font weight
* Line height
* Paragraph spacing

according to screen size and section importance.

---

## Button Typography

Buttons should:

* Feel premium.
* Have proper font weight.
* Have proper sizing.
* Be easily clickable.
* Follow modern SaaS standards.

---

# Layout Guidelines

## Container Width

* Use proper max-width containers.
* Avoid excessive stretching on large screens.
* Maintain consistent horizontal spacing.

## Vertical Spacing

* Maintain generous whitespace between sections.
* Create clear visual separation between content blocks.
* Use consistent spacing rules throughout the page.

## Responsiveness

The design must be:

* Mobile-first
* Tablet-friendly
* Desktop optimized

Every section must adapt naturally to all screen sizes.

---

# Animation Guidelines

Animations should feel:

* Smooth
* Interactive
* Natural
* Professional

Avoid:

* Excessive motion
* Overly dramatic effects
* Distracting animations

---

## Scrolling Behaviour

Scrolling should feel smooth throughout the website.

Requirements:

* Smooth anchor navigation
* Smooth section transitions
* No sudden jumps
* Natural scrolling experience

---

## Entrance Animations

Use subtle entrance animations for:

* Hero content
* Service cards
* Process steps
* Testimonials
* CTA section

Animations should:

* Fade smoothly
* Slide naturally
* Maintain performance

---

## Hover Interactions

Interactive elements should have:

* Clear hover states
* Smooth transitions
* Visual feedback

Examples:

* Cards
* Buttons
* Navigation items
* Links

---

# Mouse Interaction

Use subtle mouse interactions only where appropriate.

Examples:

* Card tilt effects
* Cursor-follow effects
* Interactive visual feedback

Rules:

* Must remain professional.
* Must not distract users.
* Must not reduce performance.
